## Installation

In hubot project repo, run:

`npm install hubot-hubot-script-autoworld --save`

Then add **hubot-hubot-script-autoworld** to your `external-scripts.json`:

```json
["hubot-hubot-script-autoworld"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
